# 🩺 Medical Expert System

A rule-based expert system built using Python that diagnoses common diseases based on user-selected symptoms.

## 🚀 Features
- GUI using Tkinter
- Simple symptom selection (user-friendly)
- Disease prediction with confidence
- Suggestions and precautions
- PDF report generation

## 🧠 Technologies Used
- Python
- Tkinter
- ReportLab

```bash
pip install reportlab
python medical_expert_premium.py
